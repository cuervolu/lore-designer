## Default Permission

Default permissions for the tracing plugin. Allows logging from frontend and reading history.

#### This default permission set includes the following:

- `allow-log_trace`
- `allow-log_debug`
- `allow-log_info`
- `allow-log_warn`
- `allow-log_error`
- `allow-get_log_history`
- `allow-clear_log_buffer`
